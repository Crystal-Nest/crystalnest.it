This mod generator is designed to significantly accelerate the setup and development of new mods, offering a customizable yet standardized template adaptable to any mod. Built on Jared's Multiloader template, it enhances the original with additional Javadocs, annotations, standards, and the Cobweb dependency, further simplifying development. This is the same tool used by us, the Crystal Nest team, to create new mods or port them to new Minecraft versions. The default license is CNCLv1, our custom (and suggested) license designed to enable open-source mod development while retaining full rights to monetization There are a couple of other licenses available, but you are free to use any license that better suits your needs. Check the toggle below only if you are a Crystal Nest member, otherwise fill in the Ownership step. Increase the window size or go to a larger device to use the mod generator! Our template works on the following few assumptions:
  • IntelliJ IDEA is the IDE of choice.
  • Cobweb is a required dependency.
  • The source code is publicly available on GitHub.
These are motivated by the following reasons:
  • IntelliJ IDEA, even the free community edition, is a powerful IDE that speeds up mod development, especially with the Minecraft Development plugin.
  • Cobweb is a mod API that reduces code duplication by providing abstractions for common tasks like game object registration and configuration handling. Check out its wiki for a complete feature overview.
  • Publishing your source code on GitHub is good practice, even with an All Rights Reserved license. It allows you to:
    • Receive PRs for new features, bug fixes, or version porting.
    • Facilitate compatibility with other mods by other modders.
    • Track development with GitHub issues and projects.
    • Create a wiki for modder and user information.
    • Set up release actions to publish your mod automatically.
    • Share your knowledge and help others learn from your solutions.
If you do not satisfy them, we strongly advise you to update your development setup. Extract the zip folder and open the contents with IntelliJ IDEA. If not done automatically, load the Gradle project. Follow the instruction on the readme to finish the setup! Still have questions? Come chat with us on Discord!
